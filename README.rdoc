== README

Application Ruby on rails de site ecommerce.

L'application permet d'ajouter des articles et d'avoir une interface de gestions des clients et des paniers.

Le client, lui, dispose d'une utilisation classique d'un site de vente en ligne. (Création de compte, modification de compte,ajout de produit, quantité, modification de panier, validation et ordre d'achat).

L'application est sécurisée via la gem sorcery qui gère les authentifications utilisateurs.
